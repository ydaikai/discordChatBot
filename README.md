# Discord Chatbot

## Description
openAIが開発した大規模言語モデルchatGPT("gpt-3.5-turbo")をディスコード上で動かすスクリプトです.

## How to use
1. API Keyの登録: Discord, openAIの両方でAPIキーを発行して環境変数として`.env`などに保存してください.
2. main.pyを起動
3. 以上.
